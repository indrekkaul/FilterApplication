# Filter App

## Kasutatud tehnoloogiad ja versioonid

- **Java**: 17
- **Spring Boot**: 3.2.7
- **Node.js**: 14.17.5
- **npm**: 6.14.14
- **React**: 17.0.2

## Projekti struktuur

- `src/`: Sisaldab Spring Boot backend'i koodi.
- `filter-app/`: Sisaldab React frontend'i koodi.

## Projekti backend käivitamiseks

```bash
./gradlew bootRun
```

## Projekti frontend käivitamiseks

```bash
cd filter-app
npm start
```
